# e-plantShopping
Paradise Nursery e-commerce app built with React, Redux Toolkit, and React Router.
E-Plant Shopping (Paradise Nursery)

E-Plant Shopping is a React and Redux-based e-commerce application for browsing and purchasing houseplants online. The application features a welcoming landing page, a categorized product listing page, and a fully functional shopping cart that allows users to add, remove, and manage plant quantities before checkout.

Features

* Attractive landing page with company information and navigation.
* Product catalog displaying houseplants organized into multiple categories.
* Add-to-cart functionality with dynamic cart updates.
* Shopping cart with quantity increase/decrease controls.
* Automatic calculation of item subtotals and total cart cost.
* Dynamic cart icon showing the current number of items.
* Redux state management for efficient cart operations.
* Responsive and user-friendly interface.

Technologies Used

* React
* Redux Toolkit
* React Router
* JavaScript (ES6+)
* CSS3

Project Structure

* Landing Page – Company introduction and entry point to the application.
* Product Listing Page – Browse categorized houseplants and add them to the cart.
* Shopping Cart Page – Manage selected items, quantities, and total cost.
